Contribute data

Small, focused data additions are perfect — one tax at a time.

Start with your local jurisdiction

Pick your city or county first. Find one tax with an official source and add it as a small JSON file via Pull Request or Issue. Get in touch if you'd like some help.

Suggest via GitHub Issue Add JSON via Pull Request

We auto-sort keys and validate JSON in CI. Small, accurate contributions are best.
